#1. Observer

- 마우스 휠, 마우스 클릭, 핸드폰 터치 이벤트에 반응

- gsap.registerPlugin(Observer);※ scrollTrigger를 이미 사용중이면 따로 Observer를 등록할 필요 없음

            Observer.create({
              target: window,
              type: "wheel, touch",
              onUp: function() {},
              onDown: function() {},
              .
              .
              .
              계속해서 함수를 추가할 수 있음
              자유롭게 마우스 이벤트 제어 가능
            });

#2. FLIP

- "First", "Last", "Invert", "Play"

- 이벤트가 발생한 요소에 다시 이벤트를 주면 이벤트가 되감기 되는 효과

- Flip.getState("클래스 명")로, 해당 클래스 명의 현재 position / size / rotation 정보를 담아서 리턴해줌

- gsap.registerPlugin(Flip);
